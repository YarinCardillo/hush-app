# E2EE Badge: Home Page Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a centered "end-to-end encrypted" badge between the subtitle and the glass form card on the home page.

**Architecture:** Single `motion.div` added to `Home.jsx` JSX, styled via an inline style entry in the existing `styles` object. Uses only existing CSS custom properties (`--hush-amber-ghost`, `--hush-amber`) so it automatically adapts to dark/light theme. Inline SVG lock icon (Lucide-style).

**Tech Stack:** React 18, Framer Motion (`motion/react`), inline styles, CSS custom properties.

---

### Task 1: Add badge style entry and JSX to Home.jsx

**Files:**
- Modify: `client/src/pages/Home.jsx`

**Step 1: Add `e2eeBadge` style entry to the `styles` object**

In `client/src/pages/Home.jsx`, find the `styles` object (starts at line 21). Add this entry after `logoSub`:

```js
e2eeBadge: {
  display: 'inline-flex',
  alignItems: 'center',
  gap: '5px',
  padding: '3px 10px',
  background: 'var(--hush-amber-ghost)',
  color: 'var(--hush-amber)',
  fontSize: '0.65rem',
  fontWeight: 500,
  letterSpacing: '0.06em',
  textTransform: 'uppercase',
  border: '1px solid transparent',
  borderRadius: 0,
  userSelect: 'none',
},
```

**Step 2: Add the badge `motion.div` between the logo block and the glass form card**

In the JSX, between the closing `</motion.div>` of the logo block (around line 560) and the opening `<motion.div` of the glass card (around line 563), insert:

```jsx
{/* E2EE badge: trust signal between logo and form */}
<motion.div
  style={{ textAlign: 'center', marginBottom: '16px' }}
  initial={{ opacity: 0, y: 4 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.35, ease: [0.16, 1, 0.3, 1], delay: 0.6 }}
>
  <span style={styles.e2eeBadge}>
    <svg
      width="10"
      height="10"
      viewBox="0 0 24 24"
      fill="none"
      stroke="currentColor"
      strokeWidth="2"
      strokeLinecap="round"
      strokeLinejoin="round"
      aria-hidden="true"
    >
      <rect x="3" y="11" width="18" height="11" rx="2" ry="2" />
      <path d="M7 11V7a5 5 0 0 1 10 0v4" />
    </svg>
    end-to-end encrypted
  </span>
</motion.div>
```
